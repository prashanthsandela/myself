# prashanthsandela.com

Personal site — a single-file, no-build GitHub Pages homepage with a
"console / instrument panel" aesthetic drawn from real-time ML serving.

- **`index.html`** — the entire site. Edit, commit, done (rebuilds in ~1 min).
- **`CNAME`** — maps the custom domain `prashanthsandela.com`.
- **`SETUP-AND-DESIGN.md`** — deploy steps and the design system. Read this
  before changing anything so the site stays coherent.

> Note: for a *user site* served at the root, GitHub expects the repo to be
> named `prashanthsandela.github.io`. See `SETUP-AND-DESIGN.md` → Part 1.
